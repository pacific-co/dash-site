Documentation

Introduction

What is Blaze?

Blaze is an expressive, fearless programming language designed for developers who want to write fast, compile to native binaries, and maintain full control over their development identity.

Built with the philosophy that your IDE should meet your identity, Blaze provides a unique development experience that combines the best of modern language design with powerful memory management capabilities.

Philosophy and Goals

  • Expressive: Clear, readable syntax that makes intent obvious
  • Fearless: Memory safety without sacrificing performance
  • Fast: Optimized compilation and runtime performance
  • Native: Direct compilation to platform-specific binaries

Getting Started

Installation

git clone zen-net/blazelang
cd blaze
python blaze.py

Creating a New Project 

blaze new my_project
cd my_project

This creates a new Blaze project with the standard directory structure and configuration files.

Compiling Your Code 

blaze compile

Compiles your Blaze code to a native binary executable.

The @ symbol creates memory references, while data() allocates managed memory for complex data structures.

Advanced Topics

Compilation to Binary

Blaze compiles directly to native machine code, providing optimal performance for your applications. 

blaze compile --release --target x86_64-linux
blaze compile --release --target arm64-macos
blaze compile --release --target x86_64-windows

File Structure

// blaze.toml
[package]
name = "my_app"
version = "0.1.0"
author = "Your Name"

[dependencies]
